# Visão Geral do Projeto

## Nome do Projeto
Residencial Novo Potengi — Site de Divulgação e Cadastro

## Contexto
O município de Potengi/CE, no âmbito do Programa Minha Casa Minha Vida (MCMV), modalidade Cidades, irá doar um terreno para a construção de unidades habitacionais destinadas à população de baixa renda. A doação do terreno será revertida em abatimento no valor das unidades para os mutuários finais.

## Objetivo do Site
Criar um site institucional de divulgação do empreendimento e captação de pessoas interessadas nas unidades habitacionais, funcionando como vitrine do projeto e canal de pré-cadastro.

## Público-Alvo
- Moradores de Potengi/CE e região interessados em adquirir imóvel pelo MCMV
- Famílias com renda familiar dentro das faixas do Programa MCMV

## Escopo Inicial
1. Página institucional de apresentação do Residencial Novo Potengi
2. Página com informações sobre o Programa MCMV modalidade Cidades
3. Página de pré-cadastro para pessoas interessadas
4. Painel administrativo básico para gestão dos cadastros

## Benefícios Esperados
- Transparência na divulgação do projeto habitacional
- Agilidade no processo de captação de interessados
- Organização centralizada dos dados dos candidatos
- Facilidade de comunicação com os pré-cadastrados
