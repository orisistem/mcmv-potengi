# Regras de Negócio

## Cadastro de Interessados

| Código | Descrição |
|--------|-----------|
| RN001 | Cada CPF só pode realizar um único pré-cadastro |
| RN002 | O candidato deve ter idade mínima de 18 anos na data do cadastro |
| RN003 | O candidato não pode possuir imóvel próprio em seu nome (ou do cônjuge) para se enquadrar nos critérios do MCMV |
| RN004 | A renda familiar mensal informada deve se enquadrar nas faixas do Programa MCMV definidas pela legislação vigente |
| RN005 | O pré-cadastro não garante a concessão da unidade habitacional — é apenas uma manifestação de interesse |
| RN006 | Documentação incompleta ou irregular pode levar à desclassificação do candidato |

## Fluxo de Status do Candidato

| Código | Descrição |
|--------|-----------|
| RN007 | O candidato inicia com status "Pré-cadastrado" após preenchimento do formulário |
| RN008 | A transição de status deve seguir o fluxo: Pré-cadastrado → Em análise → Documentação Pendente / Aprovado / Reprovado → Convocado → Contemplado / Desistente |
| RN009 | Um candidato com status "Reprovado" ou "Desistente" não pode retornar a status anterior |
| RN010 | Alterações de status devem ser registradas com data, hora e identificação do administrador responsável |

## Doação do Terreno

| Código | Descrição |
|--------|-----------|
| RN011 | O valor do terreno doado pela Prefeitura de Potengi/CE será convertido em abatimento no valor das unidades habitacionais para os mutuários finais |
| RN012 | O benefício do abatimento se aplica exclusivamente aos candidatos contemplados no processo seletivo |

## Privacidade e LGPD

| Código | Descrição |
|--------|-----------|
| RN013 | Os dados pessoais dos candidatos não podem ser compartilhados com terceiros sem autorização expressa |
| RN014 | O candidato pode solicitar a exclusão de seus dados a qualquer momento, conforme direito do titular previsto na LGPD |
| RN015 | Os dados de cadastro devem ser mantidos pelo período necessário à execução do processo seletivo e after serão eliminados conforme política de retenção |
